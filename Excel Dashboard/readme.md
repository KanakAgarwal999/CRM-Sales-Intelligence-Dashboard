# CRM Sales Intelligence Dashboard

## Overview

This Excel dashboard analyzes sales pipeline performance, customer acquisition, revenue generation, and sales representative effectiveness.

The workbook contains:

- CRM Dataset
- Dashboard
- Rep Performance Analysis
- Funnel Analysis
- Industry Analysis
- Pivot Tables

## Key Business Areas

### Sales Performance
- Revenue Tracking
- Win Rate Analysis
- Deal Performance

### Pipeline Management
- Open Opportunities
- Conversion Rates
- Sales Funnel Analysis

### Customer Analysis
- Industry Segmentation
- Customer Size Analysis
- Customer Satisfaction Metrics

### Sales Team Analysis
- Sales Representative Performance
- Won Revenue
- Average Deal Size
- Sales Cycle Analysis

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design
