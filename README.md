# 📈 CRM Sales Intelligence Dashboard

## 📌 Project Overview

A comprehensive CRM Sales Intelligence Dashboard built in Microsoft Excel to monitor sales performance, pipeline health, customer acquisition, and revenue generation.

The dashboard helps sales managers identify opportunities, improve conversion rates, and optimize sales team performance.

---

## 🎯 Business Objectives

- Monitor sales pipeline performance
- Track revenue generation
- Analyze sales representative effectiveness
- Improve conversion rates
- Identify industry-specific opportunities
- Evaluate customer satisfaction

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

---

## 📊 Key KPIs

- Total Pipeline Value
- Won Revenue
- Win Rate
- Open Deals
- Lost Deals
- Average Deal Size
- Customer Satisfaction Score
- Sales Cycle Duration

## 🔍 Key Insights

### Sales Performance

- Revenue is concentrated among a few high-performing representatives.
- Several open deals represent significant future revenue opportunities.

### Funnel Analysis

- Conversion rates decline significantly in later stages.
- Certain stages experience the highest drop-off rates.

### Industry Analysis

- Some industries generate higher revenue despite fewer deals.
- Customer segment performance varies considerably.

### Customer Satisfaction

- High CSAT scores correlate with successful deal closures.
- Shorter sales cycles often produce better customer outcomes.
